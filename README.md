# 個人房貸試算器

本專案為 Windows Forms 應用程式，用於計算房貸相關資訊，包括每月應繳金額、總利息支出與總還款金額等。

<img width="1332" height="737" alt="image" src="https://github.com/user-attachments/assets/eb9bac39-7d6c-4c10-baec-5a3ecc34c33e" />

## 📌 功能介紹

- 支援自備款「比例」或「金額」輸入
- 自動計算：
  - 貸款總金額
  - 每月應繳金額（本＋息）
  - 首期利息
  - 首期本金
  - 總利息支出
  - 總還款金額
- 支援寬限期（前幾年只繳利息）

---

## 🖥️ 使用說明

### （1）輸入項目說明

使用者需在左側輸入區填入以下資料：

- **房屋總價（元）**  
  輸入房屋價格，例如：`10000000`

- **自備款比例或金額**  
  可選擇「比例(%)」或「金額(元)」
  - 若選擇比例，輸入如 `20` 代表 20%
  - 若選擇金額，輸入實際金額

- **貸款利率（年利率 %）**  
  例如：`2.15`

- **貸款年限（年）**  
  例如：`30`

- **寬限期（年）**  
  可不填，若填入則表示前幾年只繳利息不繳本金

---

### （2）操作方式

1. 輸入所有必要資料  
2. 選擇自備款輸入方式（比例或金額）  
3. 點擊「計算」按鈕  
4. 系統會在右側顯示計算結果  

若需要重新輸入，可按「清除」按鈕清空所有欄位。

---

### （3）輸出結果說明

程式會顯示以下結果：

- **貸款總金額**：房價扣除自備款後的貸款金額  
- **每月應繳金額（本＋息）**：每月需償還的金額  
- **首期利息**：第一期所需支付的利息  
- **首期本金**：第一期償還的本金  
- **總利息支出**：整體貸款期間支付的利息總額  
- **總還款金額**：本金加上利息的總支出  

所有數值皆以**千分位與小數點兩位**格式顯示。
